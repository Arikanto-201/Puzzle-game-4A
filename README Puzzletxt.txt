Bienvenue sur le jeu puzzle à glissement.
Le but du jeu est d'arranger les nombres par ordre croissant. Vous aurez une disponibilité de glisser le nombre disponible du coté du case vide depuis une variable  "k" que vous devez remplir avant de jouer. Pour acceder au jeu nous avons ce fichier qui se nomme "Interfpuzzle" qui sera notre interface de bienvenue. Ce jeu comporte 2 tailles différents tels que la taille 3*3 et 4*4. Les conditions suivantes sont valables pour tout ces deux tailles de puzzle :
    - Vous devez tout d'abord choisir la taille du jeu que vous voulez jouer 
    - Vous devrez completer le donnée qui est la variable "k", qui sera le nombre de deplacement dont vous aurez avantage 
    - Dès que le nombre de déplacement k est atteint, vous aurez une possibilité de permutter 2 nombres qui sera appelé "swap". Vous utiliserez votre souris et cliquer le 1er nombre en superposant le clic sur la 2ème. 
    - Pour notre jeu si le swap atteint le nombre 10. Le jeu propose un resolution automatique du jeu ou bien de continuer de jouer manuellement.
On vous souhaite un bon jeu.
